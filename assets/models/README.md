# 3D-модели

Сюда клади модели: `.obj`, `.gltf`, `.glb`, `.fbx` и т.п.

`build.bat` копирует содержимое этой папки в `build/assets/models/` рядом с
`Engine.exe`. Поэтому в рантайме грузи модель по относительному пути
`assets/models/<файл>` (рабочая папка процесса = `build/`), как шейдеры грузятся
из `CompiledShaders/`.

В Visual Studio эти файлы видны в фильтре `assets/models` — это ассеты, не код,
они не компилируются.
