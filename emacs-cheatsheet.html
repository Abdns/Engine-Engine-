Emacs Cheatsheet

Emacs 30 + evil + general (SPC-leader) + corfu + eglot + tree-sitter + treemacs
Обозначения: C-x = Ctrl+x  |  M-x = Alt+x  |  S-x = Shift+x  |  SPC = пробел (в Normal mode = leader)  |  C-x C-f = Ctrl+x потом Ctrl+f  |  dd = нажать d два раза (Normal mode)

🧠 Режимы

Emacs-команды (Ctrl/Alt/Shift+что-то) — везде, всегда:
C-x C-fОткрыть файл (пример)
M-xПалитра команд
C-gОтмена ВСЕГО
SPC — leader, только в Normal/Visual:
SPC f fОткрыть файл
SPC SPCПереключить буфер
Vim (одиночные буквы) — только Normal/Visual:
hjkl dd yyVim-команды
Переключение режимов:
i / a / o→ Insert mode
Esc / jk→ Normal mode
v / V / C-v→ Visual mode
Печатаешь — буквы не появляются? Жми i
